# Guía de Contribución

Gracias por tu interés en contribuir a este proyecto enfocado en el ODS 3: Salud y Bienestar.

## 📌 Flujo de trabajo

- Crear una rama a partir de `main`
- Nombrar ramas como: feature/nombre-funcionalidad
- Realizar commits claros y descriptivos
- Enviar Pull Request para revisión
- Mantener historial limpio mediante rebase

## 🔐 Seguridad y datos sensibles

- No subir información real de pacientes
- No incluir credenciales o claves en el repositorio
- Usar datos simulados para pruebas

## ⚖️ Ética del desarrollo

- Evitar sesgos en el manejo de datos
- Garantizar accesibilidad del sistema
- Diseñar soluciones inclusivas
- Priorizar la privacidad del usuario

## ✅ Requisitos para aprobar cambios

- Cumplir con estándares de documentación
- Respetar principios éticos del proyecto
- Validación mediante pipeline CI/CD
